# Word2Vec Implementation in Numpy

This repository contains my implementation of Word2Vec in raw Numpy. The code includes both the Skip-Gram and the Continous-Bag-Of-Word (CBOW) paradigms.
The final model has been trained on the HuggingFace dataset: [Salesforce/wikitext](https://huggingface.co/datasets/Salesforce/wikitext) on HuggingFace.
